### Swagger Documentation: 

1. Clone the repo
   ```sh
   git clone https://github.com/ferferq/2challengeNodeJS.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or YARN
    ```sh
   yarn
   ```
3. start the server:
  ```sh
   yarn dev
   ```
